## 📌 Project Title
AI for Early Detection of Heart Disease using ECG and Imaging

---

## 📚 Project Type
Research Project

---

##  Description
This project focuses on early detection of heart disease using Artificial Intelligence techniques.  
A deep learning model based on ResNet50 architecture is used for classification of ECG and imaging data.

The model is trained using transfer learning and optimized through preprocessing techniques.

---

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- Deep Learning (CNN)
- ResNet50
- NumPy, Pandas, Matplotlib

---

## 📊 Results
- Achieved accuracy: **87.70%**
- ROC-AUC: **0.87 – 0.97**
- Strong classification performance across multiple heart disease classes
